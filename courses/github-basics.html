GitHub 基礎操作

GitHub 簡介

GitHub 是全球最大的程式碼託管平台，提供版本控制和協作功能，是開發者必備的工具。

版本控制

追蹤和管理程式碼的變更歷史

協作開發

多人同時開發專案的完美解決方案

程式碼託管

安全可靠的雲端程式碼儲存空間

環境設定

步驟 1：安裝 Git

首先需要在本機安裝 Git 版本控制系統

# Windows 使用者可以從官網下載安裝檔
# Mac 使用者可以使用 Homebrew 安裝
brew install git

步驟 2：註冊 GitHub 帳號

前往 GitHub 官網註冊帳號

註冊帳號

步驟 3：設定 Git

git config --global user.name "您的名字"
git config --global user.email "您的信箱"

基本指令

指令 說明 範例
git init 初始化 Git 倉庫 git init
git clone 複製遠端倉庫 git clone URL
git add 加入追蹤檔案 git add .
git commit 提交變更 git commit -m "訊息"
git push 推送至遠端 git push origin main

工作流程

1. 建立/複製專案

建立新的 Git 倉庫或複製現有專案

2. 開發功能

在本地進行程式碼的編寫和測試

3. 提交變更

將變更提交到版本控制系統

4. 推送與合併

將變更推送到遠端並進行合併

實作練習

練習 1：建立第一個 GitHub 專案

跟著以下步驟來建立和管理您的第一個 GitHub 專案：

  1. 在 GitHub 上建立新的倉庫
  2. 複製倉庫到本地端
  3. 新增一個簡單的 README.md 檔案
  4. 提交並推送變更
