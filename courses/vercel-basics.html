網域設定

使用預設網域

每個專案都會獲得一個 .vercel.app 的網域

your-project.vercel.app

自訂網域

設定自己的網域名稱

# 新增 DNS 記錄
CNAME record: cname.vercel-dns.com

部署設定

環境變數設定

管理不同環境的配置

# .env 檔案範例
API_URL=https://api.example.com
SECRET_KEY=your-secret-key

部署鉤子 (Hooks)

設定部署前後的自動化任務

# package.json
{
  "scripts": {
    "build": "next build",
    "vercel-build": "npm run build"
  }
}
